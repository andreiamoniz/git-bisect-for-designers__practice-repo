• git bisect start
  • git bisect good [sha]
  • git bisect bad
  • Now, the bisect begins.
  • Is the bug still present? If so, git bisect bad. If not, git bisect good.
  • Continue until bad commit found.
  • Copy and paste found commit sha
  • git bisect reset
  • Test to see if the bug is still present
  • git revert [sha] or fix the bug
  • You're done! Goedzo!
